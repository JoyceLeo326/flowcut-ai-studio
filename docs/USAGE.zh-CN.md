# FlowCut AI Studio 使用说明

## 1. 启动

在 PowerShell 中运行：

```powershell
cd "D:\整理后的文件_2026-05-24\04_比赛活动资料\workflow\FlowCut-AI-Studio"
.\start-web.ps1
```

浏览器打开 `http://localhost:3200`。如果这个窗口关闭，服务也会停止；再次运行脚本即可恢复。

## 2. 新建项目与导入

点击“新建项目”，进入编辑器后点击左侧“Import”。可以一次选择多个视频、图片和音频。浏览器只读取你主动选择的文件，不会扫描整个 D 盘。

## 3. 三种 AI 模式

- **本地**：只生成并执行本地可完成的步骤，素材不会上传。
- **混合**：画幅、顺排、片段收紧在本地执行；字幕、静音检测和语义精选生成 ChatCut 交接任务。
- **ChatCut**：方案主要面向 ChatCut 云端执行，适合需要语音转录或按内容选取高光的任务。

## 4. 自动剪辑

在右侧“AI 方案”中输入目标，例如：

> 删除停顿，剪成 60 秒 9:16 竖屏精华并生成字幕

点击“生成剪辑方案”。每一步都有勾选框、执行位置和状态。确认后：

- 点击“应用本地步骤”修改时间线；旁边的撤销按钮会一次撤销整组修改。
- 点击“复制 ChatCut 任务”获得包含项目、素材文件名和云端步骤的标准交接包。
- 将任务粘贴到已经启用 ChatCut 插件的 Codex 任务中，再按提示附上对应视频。

FlowCut 不会因为生成交接包而自动上传视频。真正上传到 ChatCut 前应再次确认；相关云端操作可能消耗 ChatCut 积分。

## 5. 手工精修

- 左侧：素材、声音、文字、贴纸、特效、转场、字幕和调整工具。
- 中间：视频预览，可拖拽和缩放当前元素。
- 下方：多轨时间线，可剪切、移动、调速、静音和添加关键帧。
- 右侧“属性”：当前元素的位置、大小、音量、速度、混合、遮罩和特效。

## 6. 导出

点击右上角“Export”，选择 MP4 或 WebM、质量和是否包含音频，然后再次点击“Export”。浏览器默认下载目录可能在 C 盘；如果要严格保持 C 盘零媒体占用，请在浏览器设置中把下载目录改为 D 盘，或每次导出后立即移动文件。

## 7. 当前限制

- 浏览器版不能直接读取任意本地路径，必须由用户选择文件。
- ChatCut 与 FlowCut 目前通过可审阅的 JSON 交接包和成片回传连接，不是实时双向时间线同步。
- 本地“收紧片段首尾”是确定性规则；真正的静音检测、语义精选和语音字幕由 ChatCut 执行。
- 移动端只提供受限预览，正式剪辑请使用桌面 Chrome。
